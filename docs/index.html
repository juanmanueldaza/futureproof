What is FutureProof?

A Python CLI that aggregates your professional footprint and uses AI to generate strategic career outputs.

Data Aggregation

Pull data from LinkedIn, GitHub, GitLab, and portfolio sites into normalized markdown

CV Generation

Produce ATS-optimized and creative CVs in English and Spanish (MD + PDF)

Career Analysis

Compare stated goals vs actual behavior. Identify gaps and blind spots

Strategic Advice

Get actionable recommendations powered by Gemini AI for your target role

Quick Start

$ git clone https://github.com/juanmanueldaza/futureproof
$ cd futureproof && uv sync
$ cp .env.example .env # Add your GEMINI_API_KEY
$ futureproof gather all # Gather from all sources
$ futureproof analyze full # Comprehensive analysis
$ futureproof generate cv --all # Generate all CV versions

Privacy & Security

FutureProof implements security best practices to protect your data.

  • PII anonymization before external API calls
  • Prompt injection detection and blocking
  • SSRF protection for portfolio scraping
  • Path traversal prevention
  • Input validation via Pydantic
  • Secure file permissions (0600)
