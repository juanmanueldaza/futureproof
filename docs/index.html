What is FutureProof?

A single LangChain agent with 36 specialized tools that gathers your professional footprint, indexes it into a searchable knowledge base, and uses AI to analyze your career and generate strategic outputs. Everything happens through interactive chat.

Data Gathering

Pull data from LinkedIn exports, GitHub, GitLab, portfolio sites, and CliftonStrengths PDFs

Knowledge Base

ChromaDB-powered RAG search over all your career data with semantic retrieval

Career Analysis

Skill gap identification, career alignment scoring, and strategic advice

Market Intelligence

Job search across 7+ boards, tech trends from Hacker News, salary insights

CV Generation

ATS-optimized and creative CVs in English and Spanish (Markdown + PDF)

Episodic Memory

Remembers your decisions, job applications, and preferences across sessions

How It Works

Everything happens through natural language chat. Just tell the agent what you need.

Gather

The agent collects and indexes your career data into a searchable knowledge base

"Gather all my career data"

Analyze

AI examines your profile, identifies skill gaps, and assesses market fit

"Analyze my gaps for Staff Engineer"

Generate

Create optimized CVs tailored for ATS systems or creative presentation

"Generate my CV in ATS format"

Advise

Get strategic recommendations based on market data and your career trajectory

"What should I focus on for ML Engineer?"

Quick Start

$ git clone https://github.com/juanmanueldaza/futureproof
$ cd futureproof && pip install -r requirements.txt && pip install -e .
$ cp .env.example .env # Add your Azure OpenAI keys
$ futureproof chat # Start chatting
$ futureproof ask "Analyze my skill gaps" # Quick question

Privacy & Security

FutureProof implements security best practices to protect your data.

  • PII anonymization before external API calls
  • Prompt injection detection and blocking
  • SSRF protection for portfolio scraping
  • Command injection prevention
  • Input validation via Pydantic
  • Secure file permissions (0600)
